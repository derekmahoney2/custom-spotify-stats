# custom-spotify-stats
A personal Spotify listening history analyzer built with Python, pandas, and Streamlit.

# Setup
1. Request your extended streaming history from Spotify (Account → Privacy Settings)
2. Place the downloaded folder next to 'spotify.py' and rename it 'Spotify Extended Streaming History'
3. Install dependencies: 'pip install pandas streamlit plotly'
4. Run: 'streamlit run spotify.py'

# Features
- Top artists and songs
- Listening habits by hour
- Skip analysis
- Platform breakdown
- Filters by year, platform, shuffle, and offline mode
